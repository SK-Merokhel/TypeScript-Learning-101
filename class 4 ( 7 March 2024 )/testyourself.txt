   a) Adds a value to a variable
   b) Multiplies a value with a variable

2. What will be the value of 'a' after the operation 'a += 5' if 'a' initially equals 10?
   a) 5
   b) 10
   c) 15
   d) 20

3. Which operator in JavaScript combines assignment with addition?

   a) 21
   b) 30
   c) 10
   d) 3

5. What is the purpose of the modulus operator (%) in JavaScript?
   a) Adds two numbers
   b) Divides one number by another and returns the quotient

   a) 0
   b) 1
   c) 2
   d) 3

   b) Decrements the value of 'a' before using it

   d) Increments the value of 'a' after using it

   a) 0, 1
   b) 1, 1

   d) 1, 0

    a) BMI = weight * height
    b) BMI = weight / height

    a) Multiplication
